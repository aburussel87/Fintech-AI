📘 Overview

FinGuard AI is a cutting-edge financial management tool powered by artificial intelligence, designed to help users manage their budgets, track expenses, and gain insights into their financial health. The platform offers a user-friendly interface that combines advanced data analytics and AI-driven suggestions to optimize personal financial planning and decision-making.

Key Features

💰 Budget Planner

Track and manage your budget seamlessly. Automatically categorize expenses and receive suggestions to optimize your spending.

🤖 AI Chatbot

Get personalized financial advice and budgeting tips from an AI-driven chatbot, available anytime to guide you through your financial decisions.

📊 Transaction Tracking

Monitor your transactions in real-time, with a detailed history of your income and expenses. Stay informed on your spending patterns.

👤 Profile Management

Manage your financial profile, including personal details, goals, and historical data, with the flexibility to adjust as your financial situation evolves.

📈 Statistics and Analytics

Visualize your financial trends with detailed charts and graphs, helping you make better financial decisions and track your progress over time.

🔐 Fraud Detection

Detect suspicious activity and prevent fraud with advanced AI-powered algorithms that analyze transactions in real-time and flag potential threats.

🔑 Secure Authentication with JWT

Ensure secure user authentication using JWT tokens. Each user receives a secure token for login, making the entire platform safe and protecting sensitive data.

🧠 AI Models for Financial Insights

Leverage machine learning and AI models to provide tailored financial recommendations, predict spending patterns, and optimize budget planning based on individual behaviors.

⚙️ Setup

# Clone the repo
git clone https://github.com/your-repo/finguard-ai.git

# Navigate into backend
cd backend

# Create virtual environment
python -m venv venv
source venv/bin/activate   # Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Run the app
python server.py

Ensure you have Python 3.8+ and pip installed.

For Windows users, use venv\Scripts\activate to activate the virtual environment.

To run the server, use python server.py. The app will be available at http://

📦 Requirements

Dependencies listed in requirements.txt:

  • Flask
  • Flask-Cors
  • Flask-JWT-Extended
  • scikit-learn
  • joblib
  • pandas
  • faker
  • ollama

🔌 API Endpoints

Blueprints

  • /auth – JWT login/authentication
  • /register – Create new users
  • /login – Authenticate user
  • /profile – Update or view profile
  • /payment – Transaction verification
  • /recharge – Add wallet balance
  • /chat – AI chatbot interface
  • /fraud – Anomaly detection API
  • /statement – Transaction logs
  • /budget – Budget recommendations
  • /blockchain – Blockchain operations

Sample Request

POST /login
{
  "email": "john@example.com",
  "password": "securepass"
}

GET /statement
Headers: Authorization: Bearer <JWT_TOKEN>

🚀 Deployment

  • Can be deployed on Heroku, Render, or Railway
  • For production, set secret key via environment variables
  • Ensure LLaMA model is hosted locally or via Ollama cloud
  • Use Nginx or Gunicorn for performance

📝 Notes

  • Update app.config["SECRET_KEY"] with a strong key
  • Install ollama and download the llama3.2:3b model
  • Ensure fraud_detection_model.pkl exists at project root
