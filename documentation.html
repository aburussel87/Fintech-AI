Finguard AI is a secure, intelligent, and modular fintech assistant designed to help users manage their finances through personalized insights, automated budgeting, and fraud detection. It combines AI, a robust backend, and blockchain technology to ensure transparency, efficiency, and user security.

Key Features

  • AI-Powered Financial Guidance: Understands user queries and offers real-time budgeting and spending suggestions using natural language processing (NLP).
  • Smart Budgeting Assistant: Automatically generates customized budgets based on user income and expenditure patterns.
  • Conversational Chatbot: A responsive assistant to interact with users naturally, answer queries, and provide insights.
  • Fraud Detection Engine: Uses anomaly detection models to flag suspicious transactions and improve financial security.
  • JWT-Based Security: Provides secure authentication and role-based access control using JSON Web Tokens.
  • Blockchain Integration: Ensures data integrity and auditability through decentralized storage of transactions and identity information.
  • Invoice Generation: Automatically creates PDF and JSON invoices after successful transactions.
  • Multi-Step Registration: Includes email/phone verification, profile setup, and secure password creation.
  • Responsive UI: Offers a modern interface with animated transitions, scrollable views, and mobile-friendly design.

Technology Stack

  • Frontend: HTML, CSS, JavaScript, EmailJS, LocalStorage
  • Backend: Flask / FastAPI, modular route files (auth, dashboard, payment, etc.)
  • Security: JWT authentication, input validation
  • Blockchain: Used for immutable records and decentralized identity
  • AI & ML: NLP for chatbot, budgeting AI, and fraud detection algorithms
  • Data Storage: JSON files (for user and invoice data), extendable to databases

Use Cases

  • Personal budgeting and financial management
  • Small businesses looking for smart financial tracking
  • Educational demonstrations of AI + Blockchain in fintech

Goals

  • Promote financial literacy with AI-driven support
  • Ensure transparent and secure transaction logging using blockchain
  • Detect and mitigate fraud through machine learning
  • Provide an intuitive and seamless financial assistant experience

üîå API Endpoints

  • POST /register - Register a new user
  • POST /login - Authenticate user and receive JWT
  • GET /statement - Retrieve all user transactions
  • POST /recharge - Add balance using mobile/bank
  • POST /verifyReceiver - Verify recipient before sending money
  • POST /sendMoney - Send money to another user
