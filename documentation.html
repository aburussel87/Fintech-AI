üîå API Endpoints

  • POST /register - Register a new user
  • POST /login - Authenticate user and receive JWT
  • GET /statement - Retrieve all user transactions
  • POST /recharge - Add balance using mobile/bank
  • POST /verifyReceiver - Verify recipient before sending money
  • POST /sendMoney - Send money to another user
