📘 Project Overview

FinGuard AI is an intelligent financial management tool designed to simplify budgeting and financial planning. With a blend of intuitive design and powerful backend analytics, it enables users to gain control over their financial lives. Below is a breakdown of the core features and how they benefit the user.

Track and manage your budget seamlessly. Automatically categorize expenses and receive suggestions to optimize your spending. Easily set monthly spending limits and compare actual vs. planned expenses. FinGuard AI adapts to your habits over time, making budget adjustments smarter and more personalized.

Get intelligent recommendations based on spending patterns. Our AI helps you identify unnecessary expenses and advises on better saving strategies. It learns from your financial behavior to deliver tailored alerts, such as potential overdrafts or excessive spending. Insights are delivered in plain language, making them easy to understand and act upon.

Detailed transaction tracking with category analysis. Visual charts and timelines to help users understand their financial behavior. Search, filter, and review your spending history by category, merchant, or date. Daily, weekly, and monthly summaries help you stay informed without overwhelming you.

Set financial goals such as saving for a vacation or paying off a loan. Monitor progress and get nudges to stay on track. Each goal includes a dynamic timeline and recommended monthly contributions. Get motivational tips and progress badges to encourage consistency and celebrate milestones.

Access FinGuard AI via web, iOS, and Android apps. Your data is synced in real-time across all your devices. Enjoy a consistent, responsive experience with offline mode support for key features. Data security and privacy are prioritized with encrypted syncing and biometric login options.

Divide your monthly budget into customizable categories such as food, transportation, rent, and entertainment. Set specific limits for each category and receive alerts when nearing limits. Get visual summaries to easily track which categories consume the most of your budget and adjust accordingly.

⚙️ Setup

# Clone the repo
git clone https://github.com/aburussel87/Fintech-AI.git

# Navigate into backend
cd backend

# Create virtual environment
python -m venv venv
source venv/bin/activate   # Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Run the app
python server.py

Ensure you have Python 3.8+ and pip installed.

For Windows users, use venv\Scripts\activate to activate the virtual environment.

To run the server, use python server.py. The app will be available at http://

🚀 Deployment

  • Can be deployed on Heroku, Render, or Railway
  • For production, set secret key via environment variables
  • Ensure LLaMA model is hosted locally or via Ollama cloud
  • Use Nginx or Gunicorn for performance

📝 Notes

  • Update app.config["SECRET_KEY"] with a strong key
  • Install ollama and download the llama3.2:3b model
  • Ensure fraud_detection_model.pkl exists at project root
